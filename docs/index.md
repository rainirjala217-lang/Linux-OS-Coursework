## üß≠ Project Overview

This project explores the configuration, security, and performance evaluation of a **headless Linux server** managed entirely via the **command-line interface (CLI)** over **SSH**.  
The work is structured into **seven weekly phases**, reflecting progressive learning and implementation of system administration, security hardening, and performance analysis.

Through this project, I will:

- Deploy and administer a **Linux Server** (Ubuntu Server or equivalent) without a graphical interface  
- Manage the system remotely from a **workstation client** using **SSH**  
- Implement **core and advanced security controls**, including firewall, privilege management, and intrusion prevention  
- Develop **performance monitoring and automation scripts** using Bash  
- Evaluate **system performance under multiple workloads**  
- Conduct a **comprehensive security audit** and performance optimisation analysis

The setup includes:
- **Server System:** Linux Server (headless, CLI-only)
